# Photography Portfolio Website

This project is a **responsive photography portfolio website** developed as part of the **AGNEYRA Front-End Development Internship Challenge**. The website showcases a photographer’s work through a visually appealing gallery and slideshow layout.

## 📌 Project Objective

The objective of this project is to create a modern and responsive portfolio website that allows photographers to display their photos professionally while practicing front-end layout and design techniques.

## 🚀 Features

* Carousel slideshow to highlight featured photos
* Responsive photo gallery with an **8-image grid layout**
* Profile section with a **circular profile image overlapping the slideshow**
* Layout adjustments for **mobile, tablet, and laptop screens**
* Clean and modern user interface

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and responsive layout
* **JavaScript** – Carousel slideshow functionality
